#  FBD README
<h1 align="left">FBD - Sistema básico de gestión de pacientes para personal de salud</h1>

###

<br clear="both">

<h2 align="left">Descripción</h2>

###

<p align="left">El proyecto consiste en el desarrollo de un sistema de base de datos para la gestión básica de información médica de pacientes dentro de un hospital. El sistema permitirá al personal del área de la salud (doctores y enfermeras) consultar el historial médico de los pacientes y registrar nuevas actualizaciones, como enfermedades diagnosticadas, tratamientos aplicados, ingresos a hospitalización y egresos hospitalarios.</p>

###

<h2 align="left">Motivación</h2>

###

<p align="left">La idea de este proyecto surgió a partir de la experiencia compartida por una amiga cercana que trabaja como enfermera, quien comentó que en algunos hospitales de nuestra ciudad aún se utilizan expedientes físicos para el registro de pacientes.<br><br>Esta situación provoca que el proceso de registro y consulta del historial clínico sea tardado, además de generar un alto consumo de papel, lo que dificulta la organización, el almacenamiento y la localización de la información médica.<br><br>Ante esta problemática, se propone el desarrollo de una base de datos que permita digitalizar y organizar la información médica de los pacientes, facilitando su acceso y actualización por parte del personal de salud.</p>
